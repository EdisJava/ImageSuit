C++ app created with QT5 Creator
The app try to simulate a Suite app where the user donwload images form a JSON In this route C:/Users/User/Documents/.... and do some options with the images
Options:
Download
Delete
Show image Info
Mark Image as Favorite,
Show only favorites
Search an image
Change the style you see the images
Open Image in Full Screen
